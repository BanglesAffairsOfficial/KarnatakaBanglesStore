## 🎯 DESIGN PHILOSOPHY

### Core Principles
**Modern Luxury Meets Digital Nativity**

The theme positions bangles not as traditional jewelry, but as a **contemporary lifestyle accessory** for digitally-native audiences. It combines:

- **Cyber Minimalism**: Clean, intentional spacing with technological undertones
- **Emotional Authenticity**: Celebrates individual expression and cultural pride
- **Visual Addictiveness**: Every interaction should feel smooth, premium, and shareable
- **Creator Culture**: Positions customers as tastemakers, not just buyers

### Brand Positioning
- Not "traditional Indian jewelry"
- But "luxury digital-first lifestyle brand"
- Think: intersection of Aime Leon Dore × Rhode Skin × high-end streetwear
- Targets: Instagram tastemakers, TikTok creators, fashion-forward millennials

### Emotional Journey
```
Curiosity → Discovery → Connection → Desire → Community → Purchase → Sharing
```

## ✍️ TYPOGRAPHY SYSTEM

### FONT STACK (Google Fonts + Fallbacks)

#### Heading Font: **Clash Display**
- Weight range: 400, 500, 700
- Usage: Hero headlines, category titles, brand moments
- Characteristics: Geometric, confident, modern, attention-grabbing

#### Subheading Font: **Space Grotesk**
- Weight range: 300, 400, 500, 600
- Usage: Section headings, product names, elevated UI
- Characteristics: Technological, clean, premium

#### Body Font: **Inter**
- Weight range: 300, 400, 500, 600
- Usage: All body text, descriptions, interface
- Characteristics: Highly legible, neutral, modern

#### Accent Font: **General Sans**
- Weight range: 400, 500, 600
- Usage: CTAs, labels, specialized accent text
- Characteristics: Geometric, approachable, strong

### TYPOGRAPHIC SCALE (Heading/Hero)

```
Display (Hero):    Clash Display 72px/80px, weight 700, letter-spacing -1.5px
H1 (Main Heading): Clash Display 48px/56px, weight 700, letter-spacing -0.8px
H2 (Section):      Space Grotesk 36px/44px, weight 600, letter-spacing -0.5px
H3 (Subsection):   Space Grotesk 24px/32px, weight 600, letter-spacing -0.3px
Label/Caption:     General Sans 12px/16px, weight 500, letter-spacing 0.8px
Product Title:     Space Grotesk 20px/28px, weight 600
Body (Large):      Inter 16px/24px, weight 400, letter-spacing -0.3px
Body (Regular):    Inter 14px/22px, weight 400, letter-spacing -0.2px
Body (Small):      Inter 12px/18px, weight 400, letter-spacing 0px
Button Text:       General Sans 14px/20px, weight 600, letter-spacing -0.3px
Micro:             Inter 11px/16px, weight 500, letter-spacing 0.5px
```

### LINE HEIGHT & SPACING Strategy

```
Display Text:   1.1x (tight, impactful)
Headings:       1.2x (confident, scannable)
Body:           1.6x (readable, premium feeling)
Buttons:        1.2x (centered, solid feel)
```

### Typographic Emphasis Hierarchy

```
LEVEL 1 (Maximum Attention):
  - Clash Display, size +16px, weight 700
  - Letter spacing: -1.5px
  - All caps: NO (lowercase preferred for modern feel)

LEVEL 2 (High Attention):
  - Space Grotesk, size +4px, weight 600
  - Color: Electric blue or accent

LEVEL 3 (Medium Attention):
  - Inter, size +2px, weight 500
  - Subtle color shift or border

LEVEL 4 (Baseline):
  - Inter, size normal, weight 400
```

## 📲 MOBILE UX STRATEGY

### MOBILE-FIRST PRINCIPLES

**Metric: 100% mobile revenue focus**

1. **Thumb-Friendly Navigation**
   - All interactive elements: 48px+ touch targets
   - Back buttons positioned top-left (reachable thumb zone)
   - CTAs positioned bottom-right or centered (thumb-natural)
   - 8px minimum padding between interactive elements

2. **Gesture-Based Interaction**
   - Swipe left/right: Navigate galleries, carousels
   - Swipe up: Reveal more content, modals
   - Long-press: Wishlist, share menu
   - Double-tap: Quick add to cart
   - Pull-to-refresh: Update cart, check orders

3. **Vertical Content Flow**
   - Never horizontal scrolling (except galleries)
   - Cards stack vertically
   - Full-width cards with 16px padding
   - Images maintain square or portrait aspect

4. **Loading & Performance**
   - Target: Page load < 2s (mobile 4G)
   - Image optimization: WebP with JPEG fallback
   - Lazy load below-the-fold content
   - Show skeleton loaders during image fetch

5. **Sticky Mobile Navigation**
   - Top bar: Logo (left) + menu toggle (right)
   - 56px height
   - Background: #0A0A0A with slight blur
   - Always visible on scroll up, hide on scroll down
   - Quick-access icons: Search, Cart (badge count)

6. **Mobile CTA Strategy**
   - Sticky bottom CTA button (48px, full width minus padding)
   - Position: Above safe area (not covered by bottom bar)
   - Persistent on PDP, cart, checkout
   - Animation: Float up when content scrolled past

## ✨ ANIMATION & MOTION

### CORE ANIMATION PHILOSOPHY
- **Purposeful motion**: Every animation serves UX
- **Smooth easing**: Prefer `cubic-bezier(0.23, 1, 0.320, 1)` (modern feel)
- **Performance first**: 60fps target, GPU acceleration
- **No motion sickness**: Avoid excessive parallax or flashing
- **Accessibility**: Respect `prefers-reduced-motion`

### ANIMATION LIBRARY

#### 1. Button Hover Micro-Interaction
```css
/* Primary button on hover */
Transform: translateY(-2px)
Box-shadow: 0 12px 32px rgba(0, 217, 255, 0.24)
Background-color: darken to #00B8D4
Transition: all 0.3s cubic-bezier(0.23, 1, 0.320, 1)

/* On active press */
Transform: translateY(0px)
Box-shadow: 0 4px 16px rgba(0, 217, 255, 0.12)
```

#### 2. Product Card Hover
```
Image: Transform: scale(1.04) + rotate(-0.5deg)
Brightness: from 0.95 to 1.1
Duration: 0.5s smooth
Timing: cubic-bezier(0.23, 1, 0.320, 1)

Overlay fade: Opacity 0 → 0.1
Text lift: translateY(-4px)
Shadow: Add 0 12px 32px rgba(0, 0, 0, 0.2)
```

#### 3. Page Load Animation
```
Fade-in: Opacity 0 → 1, duration 0.5s
Stagger: Children items with 80ms delay each
Hero section: Slide up from bottom (30px offset)
Timing: ease-out cubic-bezier(0.34, 1.56, 0.64, 1)
```

#### 4. Scroll-Triggered Animations (Intersection Observer)
```
Cards entering viewport:
  - Fade in + scale from 0.95 to 1
  - Duration: 0.6s
  - Stagger: 100ms per card
  - Fire once, don't repeat

Background gradients:
  - Animated color shift: -5% to +5% on hue
  - Duration: 8s continuous loop
  - Timing: linear (subtle shift)
```

#### 5. Modal/Overlay Entrance
```
Backdrop: Fade in + blur
Duration: 0.3s
Timing: ease-out

Modal:
  - Scale from center: scale(0.92) → 1
  - Fade in: 0 → 1
  - Duration: 0.4s
  - Timing: cubic-bezier(0.23, 1, 0.320, 1)

On exit (reverse):
  - Duration: 0.25s faster
  - Creates snappy close feel
```

#### 6. Skeleton Loading Shimmer
```
Background: Linear gradient (90deg)
  - #1A1A1A → #2D2D2D → #1A1A1A
  - Background-size: 200% 100%
  - Animation: shift from -100% to +100%
  - Duration: 1.5s
  - Timing: linear (continuous)
  - Iteration: infinite
```

#### 7. Add-to-Cart Success Animation
```
Button state change:
  - Icon transform: Shopping cart → Checkmark
  - Color fade: #00D9FF → #10B981 (green success)
  - Duration: 0.5s
  - Text change: "Add to Cart" → "Added!"
  - Revert after: 2s back to original state
  - Animation: Slight bounce effect on completion
```

#### 8. Floating Elements (UGC, testimonials)
```
Transform: translateY(-8px) → translateY(8px)
Duration: 3s
Timing: ease-in-out
Iteration: infinite
Variation: Stagger each card by 0.5s for depth
```

#### 9. Hover Glow Effect (Luxury Feel)
```
Box-shadow expansion:
  - Normal: 0 4px 16px rgba(0, 0, 0, 0.1)
  - Hover: 0 12px 40px rgba(0, 217, 255, 0.2)
  - Duration: 0.3s
  - Applies to: Cards, product images, CTAs
```

#### 10. Text Reveal Animation (Hero)
```
Headline: Staggered word reveal
  - Words fade in individually
  - Each word: 0.3s duration, 80ms stagger
  - Timing: cubic-bezier(0.34, 1.56, 0.64, 1)
  - Creates "reading" effect without looking robotic

Alternative: Letter-by-letter type effect for taglines
  - Each character appears
  - Duration: 1s total for 20 characters
  - Timing: linear
```
